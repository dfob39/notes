**6.**`**reduce()**`** 函数是一个内置函数，它位于 **`**functools**`** 模块中。**`**reduce()**`** 函数用于对序列中的元素进行累积操作。它接受一个函数和一个序列作为输入，并将该函数应用于序列的元素，从而将序列减少为单个值  **

```python
from functools import reduce

def add(x, y):
    return x + y

numbers = [1, 2, 3, 4, 5]
result = reduce(add, numbers)
print(result)  # 输出: 15
```
