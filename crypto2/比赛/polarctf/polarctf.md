1.beginner
是一个简单的移位求值，把flag左移10000位，相当于把flag*2**10000，以y结尾，也就是flag_shift=y mod 10**k,k为y的位数，然后就是对这个方程的求解，用中国剩余定理分成对2**k和5**k的求解，2的k次方的话因为k<10000,所以2**10000 模2**k等于0，那么就要求y%2^k=0,符号条件，所以这个对x没有限制，而另一个就是求出2^10000 模5^k的逆元c1，x = (c1*y)%5^k也就是方程的解，转成长整数得到flag

```plain
import gmpy2
from Crypto.Util.number import *
y = 16732186163543403522711798960598469149029861032300263763941636254755451456334507142958574415880945599253440468447483752611840
k = len(str(y))
print(y % (2**k))
c1 = gmpy2.invert(2**10000,5**k)
x = (c1 * y) % 5**k
print(x)
print(long_to_bytes(x))
```
