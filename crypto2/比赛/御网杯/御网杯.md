**1.攻防世界 crypto---simpleRSA**
这题对我的启发还是蛮大的，首先又学习了一种求p和q的方式，这题里面，q是r*p+38219的下一位素数，而n已知，那么我们就可以从2**15到2**16次方遍历r，r*p*p=n,这里就可以求出近似的p，再根据p和q的关系，判断q是素数就可以求出真正的q，然后求出p，那么E1就已知了

```python
for i in range(2 ** 15, 2 ** 16):
    if isPrime(i):
        q = next_prime(i * iroot(n // i, 2)[0] + 38219)
        if n % q == 0:
            print(q)
            break

```
第二个就是一个中国剩余定理，用完开方一下就可以得到E2了
第三个就是通过最大公约数求出P，然后求出Q，接着在有限域内进行开方分别求出模p和模q下的根，找到同时满足这两个式子的m，然后就得到flag
